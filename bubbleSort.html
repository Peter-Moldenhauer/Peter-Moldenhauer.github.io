Bubble Sort Algorithm

The bubble sort algorithm is one of the most well known sorting algorithms. This simple sorting algorithm repeatedly steps through a given list (array) to be sorted and it compares each pair of adjacent elements in the list and swaps them if they are in the wrong order (numeric order). The pass through the list is repeated until no more swaps are needed and the list will be left in sorted order as a result. The algorithm, which is a comparison sort, is named for the way smaller or larger elements "bubble" to the top of the list.

Although the algorithm is simple, it is too slow and impractical for most problems. Bubble sort has worst-case and average complexity both of O(n2), where n is the number of items being sorted. Overall, bubble sort should most likely be avoided in the case of large collections due to its poor efficiency. It will also be extremely inefficient in the case of a reverse-ordered collection that it is running on.

Below, I have implemented a program that demonstrates sorting an array using bubble sort.

#include <stdio.h>
#include <stdlib.h>
#include <ctype.h>
#include <string.h>
#include <math.h>  // used to round numbers

int main()
{
    // in C there is not a built in function to sort arrays so you need to create your own
    // Bubble sort function to sort random numbers in array in ascending order

    int i, temp, swapped;
    int howMany = 10;  // how many elements are in the array
    int myArray[howMany];

    // create the array
    for(i=0; i<howMany; i++){
        myArray[i] = ( rand()%25 ) + 1;  // assign each array element a random number in the range of 1 to 25
    }

    // print out the array
    printf("Original List:\n");
    for(i=0; i<howMany; i++){
        printf("%d \n", myArray[i]);
    }

    while(1){  // infinite loop, 1 is the same as true...use a break statement to stop loop

        swapped = 0; // when swapped = 0 at the end that means the list is sorted

        //main sorting algorithm
        for(i=0; i<howMany-1; i++){

            if(myArray[i]>myArray[i+1]){
                temp = myArray[i];
                myArray[i] = myArray[i+1];
                myArray[i+1] = temp;
                swapped = 1;   // whenever you swap a number it means that the list is not in order yet, so do it again
            }

        }

        // break out of infinite loop if array is completely sorted
        if(swapped == 0){
            break;
        }
    }

    // print out sorted array
    printf("\nSorted List:\n");
    for(i=0; i<howMany; i++){
        printf("%d \n", myArray[i]);
    }

    return 0;
}
