Array implementation of the Bag (ADT)

Using an array data structure to implement the Bag ADT has its pros and cons (as in when you use an array for any implemtation). One pro is that an array is the only core data structure designed to hold a collection of elements. This is compared to other data structures that are really only designed to store a single element. Another pro to using an array is that it allows for random access. So therefore, you can get to any element in constant time (Big O complexity: O(1)). However, the big down side to using an array is that it has a fixed size. There is a maximum number of elements that has to be specified when you create an array and if you want to go beyond that maximum number at run-time then you are out of luck. Furthermore, it is often the case in which you do not know how much space is needed until you are done (or until run-time).

To implement a Bag ADT using an array, lets first assume that you have the following interface (header) file named arrayBag.h. This interface file lists out all of the public functions that control how the data structure works. Below the inteface file I will show the implementation file (named arrayBag.c) that will define all of the functions in the interface file. The implementation file is what shows the actual workings of how the data structure behaves.

arrayBag.h

# ifndef ArrayBag
# define ArrayBag

# define TYPE int 

struct arrayBag {
	TYPE data [100];
	int count;
};

void initArray(struct arrayBag * b);
void addArray(struct arrayBag * b, TYPE v);
int containsArray(struct arrayBag * b, TYPE v);
void removeArray(struct arrayBag *b, TYPE v);
int sizeArray(struct arrayBag * b);

# endif

arrayBag.c
#include "arrayBag.h"
#include <assert.h>
#include <stdlib.h>
#include <stdio.h>

struct arrayBag {
	TYPE data [100];
	int count;
};

// This function allocates space for an arrayBag structure 
struct arrayBag * createArray() {
	struct arrayBag * b = malloc(sizeof(struct arrayBag));
	return b;
}		

// This function initializes the starting size of the array to 0
void initArray (struct arrayBag * b) {
	b->count = 0; 
}	

// This function adds an element to the array and then increments the count of elements in the array
void addArray(struct arrayBag *b, TYPE v) {
	b->data[b->count] = v;
	b->count++;
}

// This function returns 1 (true) if a given value is contained in the array or 0 (false) if it is not in the array.
int containsArray(struct arrayBag *b, TYPE v) {
	int i;
	for(i=0; i<b->count; i++){
		if(b->count == v)
			return 1;
	}
	return 0;
}

// This function removes a given value from the array 
void removeArray(struct arrayBag * b, TYPE v) {
	int i = 0;
	int j;
	
	assert(containsArray(b, v)); // gives an error if you try to remove something not in the array 
	
	// find the index after the element
	while(b->data[i] != v)
		i = i + 1;
	
	// slide the elements to fill the gap 
	for(j=i; j<b->count-1; j++)
		b->data[j] = b->data[j+1];
	
	b->count = b->count-1; 
}

// This function returns the size of the array (the number of elements in the Bag) 
int sizeArray(struct arrayBag *b) {
	return b->count;
}

// This function prints out all of the values that are in the Bag
void printArray(struct arrayBag * b) {
	int i;
	for(i=0; i<100; i++)
		printf("Array[i] = %d\n", b->data[i]); 
}
